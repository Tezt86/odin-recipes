# odin-recipes

The Odin Project – Foundations Project: Recipes

## Objective

Create a basic recipe website to demonstrate my current HTML knowledge.

By the end of this project, I aimed to demonstrate:

* Basic HTML elements
* Working relative links
* Ordered and unordered lists
* Writing meaningful Git commit messages

At the end of the project, I reviewed what I accomplished and reflected on what I learned and what I would do differently.

## What I Learned

During this project, I learned the fundamentals of HTML, including creating links between pages, adding images, using relative file paths, and structuring a simple website.

I also became more confident using Git and GitHub by staging changes, writing meaningful commit messages, committing regularly, and pushing changes to a remote repository.

## What I Would Do Differently

If I were to start this project again, I would commit each HTML page together with its corresponding image. Initially, I committed the HTML files first and added the image files in a later commit. I now understand that a commit should represent a single logical change, so the page and the image it depends on should be committed together.
